# DDoSakura ToolKit（墨樱工具箱）

本项目集包括了一系列工具库、与效率工具项目

主要有：

+ GoTuner（调律器）
> 命令组合库、程序调度框架  
> 支持程序并发执行、支持异步响应、支持定时调度

+ ddosakura-modules-loader（墨樱模块加载器）
> 基于 GoTuner 和插件化开发思想的模块加载框架

+ Knowledge Collection Unit（知识收集单元）
> 利用爬虫收集学习资料的框架

+ Task Dispatch Unit（任务调度单元）
> 代替人工制定计划
